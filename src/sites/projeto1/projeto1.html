O QUE É A PROPRIEDADE DISPLAY EM CSS

A propriedade display em CSS controla a forma como um elemento HTML é exibido na página. Ela define o tipo de visualização e o comportamento do elemento no fluxo da página,
afetando seu tamanho, layout e interação com outros elementos. Os valores mais comuns são block, inline e inline-block, cada um com suas características específicas.

Display Características
Block block ocupa a largura total disponível, inicia numa nova linha, e permite a definição de width, height e margin e padding em todas as direções.
Inline inline em CSS são: o elemento não inicia em nova linha, fica na mesma linha com outros elementos, ocupa apenas o espaço necessário para o conteúdo e não aceita definições de largura e altura.
Inline-Block inline-block em CSS combina características dos valores inline e block. Permite que um elemento seja exibido lado a lado com outros elementos, como um elemento inline, mas também permite que ele tenha largura e altura definidas, assim como um elemento block.
Flex flex em CSS é usado para criar um layout flexível, onde os elementos filhos (itens flexíveis) são dispostos em uma única dimensão, seja linha ou coluna. Ele permite um controle preciso do alinhamento, espaçamento e direção dos itens, facilitando a criação de layouts responsivos e adaptáveis a diferentes tamanhos de tela.
